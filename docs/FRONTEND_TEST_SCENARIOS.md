# 원클릭 클래스 프론트엔드 테스트 시나리오

## 1. 목적과 기준

- 대상: 데스크톱 웹과 모바일 웹 프론트엔드 전체 기능
- 기본 URL: `http://localhost:5174`
- 테스트 역할: 강의자, 수강생, 비회원
- 기본 브라우저: Chromium 최신 버전
- 반응형 기준: Desktop 1440x900, Tablet 1024x768, Mobile 390x844
- 우선순위: `P0` 핵심 흐름, `P1` 주요 기능, `P2` 보조 기능
- 자동화: `E2E` Playwright 권장, `CT` Vitest+RTL 권장, `Manual` 시각·외부 연동 확인

## 2. 공통 사전 데이터

| 데이터 | 값 |
|---|---|
| 강의자 계정 | `e2e@oneclick.test` / `password` |
| 수강생 계정 | 동일 계정으로 로그인 후 역할을 수강생으로 전환 |
| 모집 중 클래스 | 노션으로 시작하는 업무 자동화 |
| 진행 중 클래스 | 나만의 브랜드 만들기 |
| 신청자 | 김서연(결제대기), 이준호(결제완료), 박민지(결제완료) |
| 지원 이미지 | JPG, PNG, WEBP 각 1개, 5MB 초과 이미지 1개 |
